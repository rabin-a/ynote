# The Quiet Web

Somewhere beneath the feed there is still a *quiet web* — pages that load instantly, hold still, and ask nothing of you.

## Signs you have found it

- The page finished loading before you noticed
- Nothing moved that you did not move
- You could read the whole thing without an account

> Attention is the only currency, and the quiet web refuses to spend yours.

Build for it anyway.

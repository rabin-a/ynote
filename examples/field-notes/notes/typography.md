# Typography notes

A few rules I keep taped above the desk.

## Measure

Keep body text between **60 and 75 characters** per line. Wider and the eye loses its place returning to the left margin.

## Scale

1. Pick one ratio
2. Apply it everywhere
3. Resist the urge to add a fourth size

```css
:root { --scale: 1.25; }
```

## Contrast

Pure black on pure white is *too much*. Warm the paper, soften the ink.

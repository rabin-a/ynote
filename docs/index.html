papery

A project-based markdown editor, previewer, and exporter. One shared renderer, so the preview is the document.

Download View on GitHub

macOS (universal) · Linux (AppImage) · Windows (.msi) · not yet notarized.

# install from the Terminal — detects your OS (macOS / Linux)
curl -fsSL https://raw.githubusercontent.com/rabin-a/papery/main/install.sh | bash

One renderer

Preview and exported HTML come from the same Rust function. Markdown is never rendered in JavaScript.

Zero runtime deps

PDF via in-process Typst. No Pandoc, no LaTeX, no headless browser. Fonts and themes are bundled.

Project = folder

Plain .md files in folders you own. No database, no lock-in — git-friendly and agent-editable.

Also a CLI

The same engine is a single binary, papery, so scripts and agents get identical output.

# render, export, batch, lint — the preview is the export
papery render notes.md -o notes.html
papery export notes.md --format pdf -o notes.pdf
papery export --all --format pdf -o dist/
papery check
And an MCP server

Give an AI agent typed, path-safe access to a project with papery-mcp.
